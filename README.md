# -

维修下单管理系统  
基于 Spring Boot + Vue 构建的前后端分离 Web 应用。

该项目用于管理维修申请、审批流程及维修订单，
帮助企业实现维修流程的数字化管理。

---

## 技术栈

### 后端

- Java
- Spring Boot
- MySQL
- RESTful API
- Maven

### 前端

- Vue
- JavaScript
- HTML5
- CSS3

### 开发工具

- Git
- IntelliJ IDEA
- Node.js

---

## 系统架构

系统采用前后端分离架构：

Vue Frontend
     │
REST API
     │
Spring Boot Backend
     │
MySQL Database

---

## 功能模块

### 用户管理

- 用户登录
- 用户注册
- 权限管理

### 维修订单管理

- 创建维修订单
- 查询订单
- 修改订单状态

### 审批流程

- 提交审批
- 审批处理
- 状态跟踪

### 数据统计

- 维修订单统计
- 历史记录查询

---

## 数据库设计

主要表：

user（用户表）  
repair_order（维修订单表）  
approval_record（审批记录表）
